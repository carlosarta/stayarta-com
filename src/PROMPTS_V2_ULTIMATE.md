# Prompts V2 Ultimate - STAYArta

Este documento contiene una colección de prompts optimizados para generar componentes y funcionalidades alineadas con la marca STAYArta. Úsalos con tu asistente de IA.

## 1. Generación de UI (Retro Pop Tech)
```text
Diseña una sección de [Nombre de Sección] para STAYArta.
Estilo: Retro Pop Tech + Neon Soft 3D.
Colores: Fondo BoneWhite (#F9F7F1), Textos DarkBlue (#273F4F), Acentos Orange (#EC5228) y Turquoise (#57B4BA).
Tipografía: DM Serif Display para títulos grandes, Sans-serif geométrica para cuerpo.
Componentes:
- Botones con bordes gruesos y sombras duras (offset shadow).
- Tarjetas con esquinas redondeadas grandes (rounded-2xl) y bordes sutiles.
- Usa gradientes de neón suaves para destacar elementos clave.
```

## 2. Generación de Dashboard (Nova AI)
```text
Crea un panel de control para el asistente Nova AI.
Modo: Dark Mode forzado (Fondo Slate-900 o DarkBlue profundo).
Estilo: Cyberpunk limpio, Glassmorphism.
Elementos:
- Gráficos de líneas finas en Turquesa neón.
- Tarjetas con fondo semitransparente y borde brillante.
- Tipografía monoespaciada para datos técnicos.
```

## 3. Redacción de Contenido (Tono de Voz)

Escribe un copy para [Producto/Servicio].
Tono: Nostálgico pero futurista, confiado, ligeramente juguetón.
